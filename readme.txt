Add the awesome Photoswipe Plugin to your Wordpress Site

== Description ==

This Plugin adds automaticly to all Image Links a Class which opens the photoswipe lightbox.

== Installation ==

1. Upload `ak_photoswipe.zip` with wordpress.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Thats all!

Not yet, but I will implement this in the future.
